# Sistema de Produtos em C#

Projeto simples feito em C# com foco em Programação Orientada a Objetos.

## Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Compra de produtos com controle de estoque
- Reposição de estoque
- Alteração de valor
- Validação de dados do usuário

## Tecnologias
- C#
- .NET Console

## Objetivo
Aprender POO na prática e lógica de programação.
